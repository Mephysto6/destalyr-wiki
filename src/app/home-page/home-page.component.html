Destalyr est un jeu de rôle sur table, comme Donjons&Dragons.
Le jeu Destalyr se déroule dans un monde médiéval fantasy où la magie est courante et les problèmes nombreux.
Destalyr est le nom du continent qui contient les 6 six races connues d'humanoïdes (appelés humains),
le reste du monde est innexploré.
