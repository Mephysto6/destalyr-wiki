Les connaissances d'une créature dépendent de ses expériences.
Une personne peut apprendre de nouvelles connaissances seule ou avec un tuteur.
Les connaissances représentent l'avancée du personnage et l'impact que l'environnement a pu avoir sur lui pendant sa vie.
Cet aspect du personnage est relativement libre et il est encouragé de réfléchir à son personnage et à son histoire pour trouver les connaissances les plus appropriées. Un joueur peut inventer des connaissances ou bien s'inspirer ou se baser sur une des connaissances présentées ici et la modifier pour la rendre plus adaptée au personnage (à voir avec le MJ pour l'équilibrage).

Quelques exemples de connaissances :
- template : vision élémentaire plus puissante + savoir sur un sujet approprié
vision de feu + cuisine / forge / etc.
vision d'air + météorologie / sonnore / etc.
vision de lumière + optique / astrologie / etc.
vision de nature + botanique / élevage / etc.
vision d'eau + sailing / alchemie / etc.
vision de ténèbres + se cacher / voir dans le noir / etc.

- template : [requirement de magie majeure] point de spéc + spécialisation dans la détection/discrétion/soins liée
feu +> détection magique
air +> détection physique
lumière +> soins
nature +> discrétion magique
eau +> soins
ténèbre +> discrétion physique

- template : [requirement de magie combinée] point de spéc + spécialisation dans une sous section/aspect de cette magie
explosion +> faire chauffer l'air et sécher les choses / explosions petites et controllées
purification +> amplifier les dégâts / soigner les blessures
eau de vie +> brûlures contrôlées plus forte que "juste feu"
âme +> perturbation magique / détection émotions
tonnerre +> interaction avec les muscles / tonnerre du ciel
animaux +> appel / self-buffs
tentacules +> structure / armes
croissance +> buff / construction
glace +> formes spécifiques / construction
terre +> construction / buff
esprit +> buff / debuff / detection mémoires
illusion +> invisibilité / apparence modifiée

- template : job + stat
forgeron + force
fermier + resistence
tanneur + dextérité
livreur + agilité
professeur + perception
marchand + éloquence

- template : interactions avec certains skills
req : armes dex > accès aux skills de force avec la dex
req : armures lourdes > l'agi capée à +1 qu'indiqué par l'armure.
req : réaction magie > gagne un autre trigger pour magie ou mouvement
req : conjuration > permet de conjurer 2 petits pets au lieu d'un moyen.
