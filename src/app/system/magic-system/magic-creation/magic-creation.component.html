Création

Creation

Pour manipuler un élément, un mage doit avoir de cet élément à disposition. Si il n'en existe pas à proximité, le mage peut en créer.
La difficulté de créer un élément dépend du type de l'élément et de la taille invoquée. Un élément peut être tangible ou intangible (noté T et i respectivement).
Les éléments tangibles sont plus difficiles à invoquer, coûtent plus d'énergie, mais ne demande qu'une concentration niveau 0 pour les maintenir sous contrôle.
À l'inverse, les éléments intangibles sont plus faciles à invoquer, coûtent moins d'énergie, mais requièrent une concentration selon la taille pour maintenir l'invocation active.
De manière générale, pour les magies intangibles, la création peut être incluse dans la manipulation et ne pas nécessiter de d12.

Quand un mage décide d'invoquer un élément, il effectue un nombre de jets de sensitivité. Ce nombre est égal à 1 + sa spécialisation dans l'élément correspondant. Le mage décide de quel résultat sera appliqué. Le résultat final se place sur une échelle continue représentant la taille de l'invocation (à la discrétion du DM).

L'invocation de magies tangibles ou combinées ajoute un coût en énergie.

Prise de contrôle

Taking control

Un mage peut essayer de voler le contrôle qu'un autre mage possède sur un élément.
Il est nécessaire que le mage possède la même magie que l'élément dont il veut prendre le contrôle.
Les deux mages vont alors faire des jets de sensitivité opposés successifs.
Le mage volant le contrôle doit obtenir 3 réussites pour voler le contrôle. Le mage se défendant doit en obtenir 2 pour le garder.
Sur l'entièreté de leurs jets, les mages peuvent ajouter leur spécialisation dans cet élément une seule fois.
